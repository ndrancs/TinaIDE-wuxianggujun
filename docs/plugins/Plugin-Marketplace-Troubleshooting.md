# 插件市场排障

> 当前状态：公开仓库只保留 Android 客户端侧插件能力与市场 API 调用边界。

后端容器、数据库、管理后台、部署脚本和生产运维排障资料已经迁入私有仓库，
不再随开源 Android 项目分发。

客户端侧排查时优先确认：

1. 设置中的服务地址是否指向你自己的服务。
2. 网络请求是否能访问插件、包管理和下载相关公开接口。
3. 本地插件缓存、下载历史和安装目录是否可读写。
4. 开源版账号登录、第三方登录、激活码、会员和官方 AI 额度入口均为移除状态，
   不应再按旧商业版链路排查。

如果问题需要查看服务端日志、数据库、管理后台或部署配置，请在私有后端仓库
中处理。

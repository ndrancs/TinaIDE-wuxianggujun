
```

```

SELinux 允许了，但仍然失败。这可能是因为：
1. 文件系统挂载选项（noexec）
2. 应用沙箱限制
3. 其他安全策略

## 解决方案：使用 JNI

### 为什么需要 JNI？

Android 应用**无法直接执行**应用私有目录中的二进制文件，即使：
- ✅ 设置了可执行权限
- ✅ 使用 shell 包装器
- ✅ 使用 sh -c 间接调用

**唯一可靠的方案**：通过 JNI 加载 `.so` 并调用其中的函数。

### 我们已经有什么？

✅ **libninja_runner.so** - 已构建并在 jniLibs 中
- 导出接口：`extern "C" int ninja_run(int argc, char** argv)`
- 大小：540 KB
- 位置：`app/src/main/jniLibs/x86_64/libninja_runner.so`

❌ **libcmake_runner.so** - 未构建（best-effort 失败）
- 需要重新运行 `build-tools.ps1` 或使用其他方案

## 推荐实施计划

### 立即（今天）

1. **创建 JNI 桥接代码**
   ```cpp
   // app/src/main/cpp/ninja_jni.cpp
   #include <jni.h>
   #include <unistd.h>
   #include <vector>
   #include <string>
   
   extern "C" int ninja_run(int argc, char** argv);
   
   extern "C" JNIEXPORT jint JNICALL
   Java_com_wuxianggujun_tinaide_core_nativebridge_NinjaRunner_runNinja(
       JNIEnv* env, jobject, jstring workDir, jobjectArray args) {
       
       // 切换工作目录
       const char* cwd = env->GetStringUTFChars(workDir, nullptr);
       chdir(cwd);
       env->ReleaseStringUTFChars(workDir, cwd);
       
       // 转换参数
       int argc = env->GetArrayLength(args);
       std::vector<char*> argv(argc + 1);
       std::vector<std::string> argStrings(argc);
       
       for (int i = 0; i < argc; i++) {
           jstring arg = (jstring)env->GetObjectArrayElement(args, i);
           const char* argChars = env->GetStringUTFChars(arg, nullptr);
           argStrings[i] = argChars;
           argv[i] = const_cast<char*>(argStrings[i].c_str());
           env->ReleaseStringUTFChars(arg, argChars);
       }
       argv[argc] = nullptr;
       
       // 调用 ninja_run
       return ninja_run(argc, argv.data());
   }
   ```

2. **配置 CMake 构建**
   ```cmake
   # app/CMakeLists.txt
   cmake_minimum_required(VERSION 3.18)
   project(ninja_jni)
   
   add_library(ninja_jni SHARED ninja_jni.cpp)
   target_link_libraries(ninja_jni ninja_runner log)
   ```

3. **更新 build.gradle.kts**
   ```kotlin
   android {
       externalNativeBuild {
           cmake {
               path = file("CMakeLists.txt")
           }
       }
   }
   ```

4. **测试**
   ```kotlin
   NinjaRunner.loadIfNeeded()
   val result = NinjaRunner.runNinja(
       workingDir = buildDir.absolutePath,
       args = arrayOf("ninja", "--version")
   )
   ```

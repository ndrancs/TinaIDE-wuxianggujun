## 常见问题

### Q: 构建工具链失败

确保 Docker 正在运行，并且有足够的磁盘空间（至少 20GB）。

### Q: 应用启动后崩溃

检查 Logcat 日志：
```bash
adb logcat -s TinaIDE CrashHandler
```

### Q: 编译失败，找不到头文件

确保已执行同步脚本：
```powershell
pwsh ./tools/sync-llvm-build.ps1 -Abi arm64-v8a -ApiLevel 24
```

### Q: 代码补全不工作

1. 确保 `compile_commands.json` 存在
2. 检查 LSP 是否初始化成功
3. 查看日志：`adb logcat -s LspService SimpleLspClient`

### Q: 模拟器上无法运行

确保使用 x86_64 架构的工具链：
```powershell
pwsh ./docker/llvm-build/build-local.ps1 -Abi x86_64 -ApiLevel 24
pwsh ./tools/sync-llvm-build.ps1 -Abi x86_64 -ApiLevel 24
```

### Q: 如何查看详细日志

```bash
# 查看所有 TinaIDE 相关日志
adb logcat | grep -i tina

# 查看编译日志
adb logcat -s NativeCompiler ClangCompiler

# 查看 LSP 日志
adb logcat -s LspService SimpleLspClient ClangdServer
```

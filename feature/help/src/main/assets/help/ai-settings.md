# AI 设置

开源版默认使用 **Custom BYOK**：你需要在本地添加自己的 AI 服务商地址、模型和 API Key。

## 渠道管理

AI 设置页支持：

- 新增、编辑和删除本地渠道
- 选择当前使用的渠道
- 在新增/编辑渠道时获取模型列表并选择模型，也可以手动输入模型名称
- 保存系统提示词、温度、上下文长度等本地参数

API Key 保存在本机配置中，不做云端同步。

## 常见问题

### 模型列表加载失败

先检查当前填写的 Base URL、API Key、模型接口兼容性和网络连通性。

### 是否支持额度或兑换码

不支持。当前公开版本只保留本地 BYOK 使用方式。
